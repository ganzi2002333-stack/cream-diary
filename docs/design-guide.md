## 6. 交互规范

### 6.1 触摸目标
- 所有可点击元素最小尺寸：**44×44px**（符合 iOS HIG）
- 标签胶囊高度：**36px**

### 6.2 动画
| 动画 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 页面切换 | 250ms | ease-out | 淡入淡出 |
| 按钮点击 | 150ms | ease | 缩放 0.97 → 1.0 |
| 卡片出现 | 300ms | ease-out | 上移 + 淡入 |
| 标签选中 | 200ms | ease | 背景色切换 + 微缩放 |
| 滑动条 | 0ms（实时） | linear | 跟手拖动 |

### 6.3 反馈
- **点击反馈**：按钮缩放 + 颜色加深
- **保存成功**：顶部弹出 Toast（"💛 已记录今日心情"），2 秒后自动消失
- **操作确认**：删除等危险操作弹出确认对话框
- **空状态**：可爱插画 + 引导语（如 "今天还没有记录哦~ 点击开始"）

## 7. 组件规范

### 7.1 情绪滑动条
```
[😔 很糟糕] ──●────────── [😊 很棒]
         1.0    当前: 6.8     10.0
```
- 滑动条高度：6px
- 滑块直径：28px（圆形，白色+阴影）
- 已填充轨道色：淡粉 `#FFE4E1`
- 未填充轨道色：`#E8D5B7`
- 当前分数使用大号字（32px Bold）居中显示在滑动条上方

### 7.2 标签胶囊
```
┌──────────┐ ┌──────────┐ ┌──────────┐
│ 😊 开心  │ │ 😢 难过  │ │ ✨ 自定义 │
└──────────┘ └──────────┘ └──────────┘
```
- 默认：浅咖背景 `#F5EDE0`，圆角 16px
- 选中：淡粉背景 `#FFE4E1`，边框浅咖
- 标签前带 emoji 图标

### 7.3 每日三问输入框
```
┌──────────────────────────────┐
│ 💭 今天发生了什么？          │
│                              │
│ （点击开始输入...）          │
│                              │
└──────────────────────────────┘
```
- 问题文字 16px SemiBold
- 输入区最小高度 80px，自动扩展
- 背景色奶黄 `#FFF8DC`，圆角 12px
- 占位符颜色 `#C4B5A5`

### 7.4 底部导航栏
```
    🏠        📅        📊        🔍        ⚙️
   今天     历史     趋势     搜索     设置
```
- 高度：60px + safe-area
- 背景：奶油白 `#FFFAF0`
- 选中项：淡粉高亮 + 加粗
- 图标使用 emoji

## 8. 页面布局

### 8.1 整体布局
```
┌────────────────────────────┐
│     状态栏 (safe-area)     │
├────────────────────────────┤
│                            │
│   页面标题 (24px Bold)     │
│                            │
├────────────────────────────┤
│                            │
│                            │
│   主内容区域                │
│   (overflow-y: auto)      │
│                            │
│                            │
├────────────────────────────┤
│   底部导航栏                │
│   🏠 📅 📊 🔍 ⚙️          │
└────────────────────────────┘
```

### 8.2 适配策略
- 最大宽度：430px，居中显示
- viewport: `width=device-width, initial-scale=1.0, viewport-fit=cover`
- 安全区域：使用 `env(safe-area-inset-*)` 适配刘海屏
