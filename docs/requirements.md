# 产品需求规格说明书

> 项目：奶油日记（Cream Diary）
> 版本：v1.0
> 日期：2026-06-19

---

## 1. 产品概述

**奶油日记**是一款运行在 iPhone 上的情绪记录 PWA 应用。用户每天记录自己的情绪状态，通过打分、标签和三问反思，追踪情绪变化趋势，培养情绪觉察能力。

### 1.1 目标用户
- 希望关注自身情绪健康的个人用户
- 偏好可爱、温馨风格界面的用户
- 非技术背景的普通用户

### 1.2 产品定位
- 个人情绪日记工具
- 数据本地存储，隐私优先

---

## 2. 功能需求

### 2.1 情绪打分（P0）
| 项目 | 说明 |
|------|------|
| 交互方式 | 滑动条（Slider） |
| 取值范围 | 1-10（连续值，保留 1 位小数） |
| 两端标注 | 左侧"很糟糕"，右侧"很棒" |
| 当前位置 | 实时显示分数及对应表情 |

### 2.2 标签系统（P0）
| 项目 | 说明 |
|------|------|
| 预设标签 | 开心、难过、焦虑、平静、愤怒、疲惫、期待、满足、孤独、感激、紧张、无聊、兴奋、困惑、自信 |
| 自定义标签 | 用户可创建新标签，名称不超过 20 个字符 |
| 选择方式 | 多选，点击切换选中/取消 |
| 标签颜色 | 每种标签带柔和色块区分 |

### 2.3 每日三问（P0）
| 序号 | 问题 | 输入方式 |
|------|------|----------|
| 第1问 | 今天发生了什么？ | 多行文本输入框，不限字数 |
| 第2问 | 我今天最明显的感受是什么？ | 多行文本输入框，不限字数 |
| 第3问 | 这件事反映了我什么需求？ | 多行文本输入框，不限字数 |

### 2.4 搜索与筛选（P1）
| 筛选维度 | 说明 |
|----------|------|
| 关键词搜索 | 在三问答案中全文搜索匹配 |
| 日期范围 | 选择开始日期 - 结束日期 |
| 情绪分数 | 滑动条设定最低分 - 最高分范围 |
| 标签筛选 | 选择标签，显示包含该标签的所有记录 |

### 2.5 情绪趋势图表（P1）
| 项目 | 说明 |
|------|------|
| 图表类型 | 折线图 |
| 时间维度 | 按周 / 按月 切换 |
| 数据点 | 每天一条（当天情绪分数） |
| Y 轴 | 1-10 分 |
| X 轴 | 日期 |

### 2.6 每日提醒（P2）
| 项目 | 说明 |
|------|------|
| 提醒方式 | Web Notification API 推送通知 |
| 默认时间 | 每晚 21:00 |
| 自定义 | 可在设置中修改提醒时间 |
| 开关 | 可开启/关闭提醒 |

### 2.7 数据备份（P1）
| 项目 | 说明 |
|------|------|
| 导出 | 导出为 JSON 文件，可保存至 iCloud Drive / 文件 App |
| 导入 | 从 JSON 文件恢复数据 |
| 格式 | 标准 JSON，包含所有记录、标签、设置 |

## 3. 非功能需求

| 类别 | 要求 |
|------|------|
| 平台 | iOS Safari（PWA 模式，支持添加到主屏幕） |
| 离线 | 通过 Service Worker 支持离线访问（首次加载后缓存） |
| 隐私 | 所有数据存储在浏览器 IndexedDB，不上传任何服务器 |
| 响应式 | 针对 iPhone 竖屏优化（375-430px 宽度范围） |
| 触摸友好 | 按钮/交互区域 ≥ 44px（符合 iOS HIG） |
| 性能 | 页面加载 < 3 秒，交互响应 < 100ms |

---

## 4. UI 设计方向

| 维度 | 规范 |
|------|------|
| 主色调 | 奶油色系：米白 #FFFAF0、奶黄 #FFF8DC、淡粉 #FFE4E1、浅咖 #D2B48C |
| 圆角 | 大圆角卡片（16-20px），按钮圆角 12px |
| 字体 | 系统默认中文字体，标题加粗 |
| 氛围 | 可爱、温馨、柔和，带有轻柔过渡动画 |
| 图标 | 手绘风格/可爱风格 emoji 为主 |
