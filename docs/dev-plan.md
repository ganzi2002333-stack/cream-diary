## 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| iOS Notification 权限难获取 | 提醒功能不可用 | 备选：微信通知/快捷指令提醒（v1.1） |
| GitHub Pages 被墙 | 国内无法访问 | 备选：Gitee Pages 或 Vercel 部署 |
| IndexedDB 数据被清理 | 用户数据丢失 | 强调定期导出 JSON 备份 |
